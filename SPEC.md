# OWS Specification Entry Point

This repository's authoritative specification is split by concern:

- [Data model](docs/core/DATA_MODEL.md)
- [Event schema](docs/core/EVENT_SCHEMA.md)
- [Package format](docs/core/PACKAGE_FORMAT.md)
- [Architecture](docs/core/ARCHITECTURE.md)
- [Threat model](docs/core/THREAT_MODEL.md)

OWS v0 is text-first, filesystem-scoped, IDE-agnostic, and locally verifiable.
Binary files are opaque/hash-only. Hosted management layers are separate future
projects.
