# Start Here

Open Work Standard (OWS) is a local-first proof-of-work protocol and toolchain. The smallest useful path is:

```text
ows init → work normally → ows package → offline verify / inspect / report
```

Event presence is evidence of recorded activity. Event absence is not proof of misconduct.
